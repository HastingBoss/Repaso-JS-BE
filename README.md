# Repaso-JS-BE 📚

Material de repaso de JavaScript para el backend, organizado por temas con ejemplos prácticos.

## 📁 Estructura

```
Clase-1/
├── Notas-md                          ← Resumen con descripción de cada tema
└── ejemplos/
    ├── 01-tipos-de-datos.js
    ├── 02-comparadores.js
    ├── 03-operadores-logicos.js
    ├── 04-operadores-aritmeticos.js
    ├── 05-bucles.js
    ├── 06-funciones.js
    ├── 07-arrays.js
    ├── 08-objetos.js
    ├── 09-metodos-arrays-basicos.js
    └── 10-metodos-arrays-avanzados.js
```

## 🚀 Cómo usar los ejemplos

Necesitás tener [Node.js](https://nodejs.org) instalado. Luego podés ejecutar cualquier archivo desde la terminal:

```bash
node Clase-1/ejemplos/01-tipos-de-datos.js
```
